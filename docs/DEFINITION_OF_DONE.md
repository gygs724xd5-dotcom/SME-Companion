# Definition Of Done

A sprint is done only when:

- Planner decisions are visible in developer diagnostics.
- High-confidence planner routes cannot fall through to generic fallback.
- Business context is stored and reused in later turns.
- Response guard prevents repeated assistant replies.
- Manual store data is isolated by authenticated owner.
- OCR and translation foundations do not fabricate provider output.
- Architecture documentation is updated.
- Verification commands are run and results recorded.
