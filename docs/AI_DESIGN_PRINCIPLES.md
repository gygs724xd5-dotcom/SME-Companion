# AI Design Principles

- Ask one specific missing question at a time.
- Never ask what is already known in business memory.
- Continue active workflows when the user provides numbers or partial answers.
- Treat planner output as authoritative routing.
- Treat LLM output as language polish only.
- Avoid generic fallback when intent confidence is medium or high.
- Use business context in the next turn after it is provided.
- Prefer deterministic workflow responses for operational tasks.
