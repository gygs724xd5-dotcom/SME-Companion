# Changelog

## V1.7 Interactive Demo Experience

- Added six demo stores for Thai SME scenarios: coffee shop, restaurant, clothing store, beauty store, construction materials, and online store.
- Added landing page flow for selecting demo stores and entering manual store setup.
- Polished Thai UX across demo cards, banners, welcome messages, content labels, and reliability messages.
- Added LLM provider layer for OpenAI and DeepSeek-backed responses with deterministic fallback.
- Added DeepSeek demo chat path for demo mode.
- Added daily and monthly budget guard for LLM usage.
- Added demo token guard to prevent demo overuse.
- Added graceful fallback behavior for missing keys, provider failures, and demo/budget exhaustion.
- Completed RC smoke, UX, demo store, AI conversation, budget/reliability, and deploy readiness audits.
